# Методичка — «Космос» (Python + pygame)

Аудитория: дети ~13 лет, **прошли «Змейку», «Ловкого кота» и «Арканоид»**.
5 уроков по ~45 мин. Четвёртая игра. Главная новая идея — **список объектов**:
пуль и врагов на экране много сразу, держим их в списках и каждый кадр
обрабатываем циклом (`append` → `for` → `remove`). Игра целиком на фигурах,
ассеты не нужны — каждый файл запускается сам.

Запуск:

```
python games/space/code/step1_ship.py
```

---

## Урок 1 — Корабль (`step1_ship.py`)
**Повторение из «Арканоида»:** движение влево-вправо, `get_pressed()`,
удержание в границах. Новое — мелочь: `draw.polygon` по списку точек (треугольник),
и поэтому `ship_x` теперь **центр**, а не левый край.

## Урок 2 — Выстрел (`step2_shoot.py`) ⭐ ключевой
**Новое — главная идея курса:** список `bullets`. Выстрел — `append`, движение —
`for b in bullets[:]`, уход за экран — `remove`. Три обязательных разговора:
- **`KEYDOWN` vs `get_pressed()`** — почему стреляем по событию (одно нажатие =
  одна пуля), а не каждый кадр.
- **`bullets[:]` (копия)** — почему нельзя удалять из списка, по которому идёшь.
- **пуля = `Rect`** — как кирпич в Арканоиде, удобно двигать и сталкивать.

## Урок 3 — Враги (`step3_enemies.py`)
**Новое:** второй список `enemies` — тот же шаблон, что и пули, но добавляет их
**сама игра** по таймеру (`spawn_timer` / `SPAWN_EVERY`) и в случайном месте
(`random.randint`). Подчеркните: код врагов почти копия кода пуль — в этом сила
приёма. `draw_enemy()` — функция-помощник (как `make_bricks` в Арканоиде).

## Урок 4 — Попадания (`step4_hits.py`)
**Новое:** двойной `for` (каждая пуля × каждый враг), `colliderect`, удаление
обоих, `score`. Ключевые ловушки: **`break`** после попадания и проход **по
копиям** обоих списков. Дайте сломать специально — «list index out of range».

## Урок 5 — Жизни и конец (`space.py` = финал)
**Новое:** `lives`, `state = "play"/"over"`, враг отнимает жизнь (прорвался вниз
ИЛИ врезался в корабль), `new_game()`, рестарт по пробелу. Тонкость: **пробел
делает два дела** — стреляет в игре и перезапускает после конца (ветка по
`state`). Плюс полный экран (F / ESC). Файл `step5_lives.py` идентичен `space.py`.

## Советы
- **Список объектов** — сердце курса. Покажите на доске: один ящик (список), в
  него кладём и из него вынимаем карточки (объекты). Пули и враги — одинаковый
  приём, только источник разный (игрок / таймер).
- **Копия списка `[:]`** — главная новая ловушка после Змейки. Стоит времени:
  пусть уберут `[:]` и увидят, как элементы «проскакивают».
- **`break` в двойном цикле** — повод повторить, что `break` выходит только из
  **внутреннего** цикла.
- **`KEYDOWN` для выстрела** — хороший контраст с `get_pressed()` для движения.
  Уберите `KEYDOWN`, поставьте `get_pressed()` — увидят «лазерный шланг».
- **Чего избегаем:** классы, `pygame.sprite.Group`, векторная математика. Всё на
  простых списках, переменных и `Rect` — как в прошлых играх.
- **Расширения** (рост сложности, звук из «Ловкого кота», рекорд, бонусы) — в
  конце урока 5.
