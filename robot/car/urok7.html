Первый заезд из pygame

Цель урока Написать первый «пульт» на Python: окно pygame ловит стрелки и командует машиной. Машина едет с клавиатуры ноутбука.

🔗 Подключение — объект car

Связь спрятана в robot.py (мы видели его в уроке 4). Одна строчка — и у нас есть машина-объект с понятными командами:

from robot import connect
car = connect()            # найдёт машину (USB/Bluetooth)
car.forward(200)         # вперёд
car.stop()               # стоп
robot.py (связь + объекты)
💡 Машины ещё нет? Работаем в «режиме показа» Если машина не подключена, robot.py сам включает режим показа: команды просто игнорируются, а датчики выдают данные понарошку. Программа не падает — можно писать и пробовать код заранее, даже без машины.

🎮 Пульт: стрелки → команды

Знакомый игровой цикл: каждый кадр смотрим, какие стрелки зажаты, и вызываем нужную команду машины. Отпустил все — stop():

keys = pygame.key.get_pressed()
if   keys[pygame.K_UP]:    car.forward(SPEED)
elif keys[pygame.K_DOWN]:  car.back(SPEED)
elif keys[pygame.K_LEFT]:  car.left_turn(SPEED)
elif keys[pygame.K_RIGHT]: car.right_turn(SPEED)
else:                     car.stop()        # отпустил — стоп

Запусти drive_keys.py: ↑↓ едем, ←→ поворот, отпустил — стоп, ESC — выход.
