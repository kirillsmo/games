Первый заезд из pygame

Цель урока Написать первый «пульт» на Python: окно pygame ловит стрелки и шлёт машине команды по Bluetooth. Машина едет с клавиатуры ноутбука.

🔗 Библиотека связи car_link.py

Связь с машиной спрятана в готовый файл car_link.py — как robot_link.py у руки-робота, только умеет ещё и читать ответы машины. Пользоваться просто:

from car_link import connect
car = connect()            # сам найдёт машину (Bluetooth/USB)
car.send("F,200")        # вперёд на скорости 200
car.send("S")            # стоп
line = car.read()         # строка телеметрии или None
car_link.py (связь)
💡 Машины ещё нет? Работаем в «режиме показа» Если машина не подключена, car_link сам включает режим показа: команды просто игнорируются, а телеметрия выдаётся понарошку. Программа не падает — можно писать и пробовать код заранее, даже без машины.

🎮 Пульт: стрелки → команды

Знакомый игровой цикл: каждый кадр смотрим, какие стрелки зажаты, и шлём команду. Хитрость — шлём её только когда она поменялась, чтобы не заваливать машину одинаковыми строками:

keys = pygame.key.get_pressed()
if   keys[pygame.K_UP]:    drive("F")
elif keys[pygame.K_DOWN]:  drive("B")
elif keys[pygame.K_LEFT]:  drive("L")
elif keys[pygame.K_RIGHT]: drive("R")
else:                     drive("S")   # отпустил — стоп

Запусти drive_keys.py: ↑↓ едем, ←→ поворот, отпустил — стоп, ESC — выход.
