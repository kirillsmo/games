Команды моторам

Цель урока Научить машину слушать команды по Serial-порту: написать буквы — и машина едет вперёд/назад/поворачивает. Пока по USB-проводу, через Serial Monitor.

🧠 Идея: команды одной буквой

Договоримся об «языке» команд. Каждая команда — одна строка:

КомандаЧто делает
F,200вперёд на скорости 200 (0…255)
B,200назад
L,200поворот налево (танком)
R,200поворот направо
Sстоп

🔄 Поворот «танком»

Чтобы повернуть налево, левые колёса крутятся назад, а правые — вперёд. Машина разворачивается на месте. В коде это две функции: одна на левые моторы, другая на правые, каждой говорим направление и скорость:

void doCommand(char dir, int speed) {
  if (dir == 'F') { leftMotors(1, speed);  rightMotors(1, speed); }
  else if (dir == 'L') { leftMotors(-1, speed); rightMotors(1, speed); }
  // ... B и R — наоборот, S — стоп
}

📥 Заливаем прошивку

Залей serial_drive.ino. Открой Serial Monitor в Arduino IDE, поставь скорость 9600 и впиши команду, например F,200, нажми Enter — машина поедет. S — стоп.

⬇️ serial_drive.ino
💡 Это тот же serial, что у руки-робота Рука MeArm слушала команды "номер,угол". Машина слушает "буква,скорость". Приём один и тот же: Arduino читает строку из порта и выполняет. В следующем уроке этот порт станет беспроводным.
✅ Проверка (машина на коробке!) Команды F, B, L, R, S из Serial Monitor работают, скорость меняет резвость колёс. Дальше — режем провод.
