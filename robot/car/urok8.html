Кокпит — приборная панель

Цель урока Превратить простой пульт в «кокпит»: добавить регулировку скорости, приборы на экране (скорость полоской, текущая команда) и индикатор связи.

⚙️ Скорость на лету

Заведём переменную speed и будем менять её клавишами + и -. Меняем по событию нажатия (а не каждый кадр), с ограничением 80…255:

if event.type == pygame.KEYDOWN:
    if event.key in (pygame.K_EQUALS, pygame.K_KP_PLUS):
        speed = min(255, speed + 20)   # быстрее
    if event.key in (pygame.K_MINUS, pygame.K_KP_MINUS):
        speed = max(80, speed - 20)    # медленнее

📊 Приборы на экране

Рисуем «полоску» скорости — это просто прямоугольник, ширина которого равна скорости. И показываем, есть ли связь (у car есть подсказка car.online):

# полоска скорости
pygame.draw.rect(screen, (60,60,80), (24,140,255,20))
pygame.draw.rect(screen, (108,178,255), (24,140, speed, 20))

link = "есть связь" if car.online else "режим показа"

Запусти cockpit.py: ↑↓ ←→ руль, +/- скорость.

⬇️ cockpit.py car_link.py
💡 Это та же приборная панель, что в играх Полоска скорости — как полоска жизней в «Космосе»: прямоугольник, длина которого зависит от числа. Ничего нового — просто рисуем по числам.
✅ Проверка На экране видно команду, скорость-полоску и состояние связи. +/- реально меняют резвость машины. Готов к самому зрелищному уроку — радару.
