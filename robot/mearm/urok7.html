Заливаем Firmata

Цель урока Один раз залить в Nano готовую прошивку Firmata — и больше Arduino программировать не нужно. Дальше всем управляет Python.

🤔 Что такое Firmata и зачем она

Можно было бы под каждый робот писать свою программу для Arduino и придумывать «протокол» команд. Но это сложно и не нужно. Есть готовая прошивка Firmata: заливаешь её в плату один раз — и Arduino превращается в «послушные ножки», которыми командует компьютер.

💡 Главная мысль курса Весь «ум» робота будет жить в Python, который ты уже знаешь по играм. Arduino — просто исполнитель: Python говорит «поставь ножку 9 на угол 120°», Firmata это делает.

📚 Шаг 1 — поставь библиотеку FirmataExpress

В Arduino IDE: Инструменты → Управление библиотеками…, найди FirmataExpress (автор MrYsLab) и нажми «Установить». Вместе с ней поставится библиотека Servo.

⬆️ Шаг 2 — залей готовый пример

Файл → Примеры → FirmataExpress → FirmataExpress

Откроется готовый скетч — писать ничего не надо. Выбери:

1. плата: Arduino Nano→ 2. порт (обычно /dev/ttyUSB0)→ 3. «Загрузить» ⬆

На Ubuntu у Nano порт обычно /dev/ttyUSB0 (чип CH340). Если порт «серый» — см. урок 8 (права доступа и драйвер CH340). Если FirmataExpress не ставится, подойдёт StandardFirmata (Файл → Примеры → Firmata → StandardFirmata) — для серво её хватает.

💡 Это нужно сделать только раз После заливки Firmata живёт в плате постоянно. Менять программу робота мы будем в Python, не трогая Arduino. Удобно: один раз залил — и забыл.

✅ Шаг 3 — проверка из Python

Поставь библиотеку Python (один раз): pip install pymata4. Затем запусти центровку — все 4 серво встанут в 90°:

python center.py
⬇️ center.py как залить Firmata

Серво дёрнулись и встали ровно, Nano не перезагрузился — значит схема и прошивка работают! Делай центровку перед тем, как надевать качалки (см. урок 2).

🎮 Попробуй Открой center.py и поменяй угол 90 на 60 для одного сустава. Запусти снова — сустав встанет иначе. Так в Python мы и будем «командовать» роботом.

🧠 Проверь себя

Выбери ответ. Ошибёшься — 30 секунд на то, чтобы вернуться в урок и перечитать.

Сколько раз заливают Firmata в плату?

✅ После заливки Firmata живёт в плате постоянно. Программу робота меняем в Python, не трогая Arduino.

Какую библиотеку Arduino ставят для серво и ультразвука?

✅ FirmataExpress (автор MrYsLab) ставится в Arduino IDE; вместе с ней — библиотека Servo.
