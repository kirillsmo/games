ПРОШИВКА ARDUINO ДЛЯ РОБОТА MeArm — теперь это стандартная Firmata.
====================================================================

Раньше под каждый робот писали свою прошивку и свой «протокол» команд. Больше так
не делаем. Заливаем в плату ОДИН раз готовую прошивку FirmataExpress — и дальше вся
логика живёт в Python (файл arm.py и программы уроков). Своих .ino-файлов больше нет.

ЧТО СДЕЛАТЬ ОДИН РАЗ
-------------------
1. Открой Arduino IDE.
2. Меню «Инструменты → Управление библиотеками…», найди и установи библиотеку
   «FirmataExpress» (автор MrYsLab). Вместе с ней поставится «Servo».
3. Меню «Файл → Примеры → FirmataExpress → FirmataExpress».
4. Выбери плату «Arduino Nano» и нужный порт (обычно /dev/ttyUSB0, чип CH340).
5. Нажми «Загрузить» (→). Всё — плату больше трогать не нужно.

   Если FirmataExpress не ставится, подойдёт и обычная «StandardFirmata»
   (Файл → Примеры → Firmata → StandardFirmata) — для сервоприводов её хватает.

ПОДКЛЮЧЕНИЕ СЕРВОПРИВОДОВ (прямо к ножкам Nano)
-----------------------------------------------
   база   -> D9
   плечо  -> D10
   локоть -> D11
   захват -> D6
   у всех серво: красный -> 5V, коричневый/чёрный -> GND.
Номера ножек заданы в arm.py (SERVO_PINS) — поменяй там, если подключил иначе.

ПИТАНИЕ
-------
4 металл-серво едят немало тока. По USB (≈0,5 А) двигай суставы ПО ОДНОМУ и плавно;
надёжнее — USB-зарядка/хаб на 2 А. Если Nano перезагружается или серво дёргаются —
не хватает тока: запитай серво от внешних 5 В, и ОБЯЗАТЕЛЬНО соедини «землю» (GND)
внешнего питания с GND платы.

ПРОВЕРКА И ЦЕНТРОВКА
-------------------
Никаких отдельных тестовых скетчей не нужно — всё из Python:
   python center.py     -> поставит все 4 серво в 90° (делай это ПЕРЕД установкой качалок!)
   python control_one.py -> покрутить один сустав стрелками
