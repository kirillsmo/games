Цель урока · ФИНАЛ Запрограммировать робота на задачу: взять предмет и переставить в другое место.

🤖 Программа = список поз

Соберём последовательность поз: подъехать → открыть → опуститься → закрыть (схватить) → поднять → перенести → опустить → открыть (отпустить) → домой. Между позами — пауза, чтобы робот успел доехать.

# [база, плечо, локоть, захват]
PROGRAM = [
    [40, 60, 60, 90],   # опустились, захват открыт
    [40, 60, 60, 30],   # сжали — взяли
    [40, 110, 110, 30], # подняли
    [140, 110, 110, 30],# перенесли
    [140, 60, 60, 90],  # опустили и отпустили
]

for pose in PROGRAM:
    for joint in range(4):
        robot.send(joint, pose[joint])
    time.sleep(1.0)        # пауза, чтобы доехать

✅ PROGRAM — список из поз: опуститься → схватить → поднять → перенести → отпустить.
