Первый скетч

Цель урока Залить первый скетч на Arduino и повернуть один сустав. Понять Servo-библиотеку.

🧰 Arduino IDE — три шага

1. выбрать плату (Uno)→ 2. выбрать порт→ 3. нажать «Загрузить» ⬆

На Ubuntu порт обычно называется /dev/ttyACM0. Если порт «серый» (не выбирается) — см. урок 8, там про права доступа.

📝 Скетч: один сустав на угол

#include <Servo.h>
Servo servo;

void setup() {
  servo.attach(9);      // серво подключено к пину 9
  servo.write(120);     // повернуть на 120°
}

void loop() { }
⬇️ one_servo_test.ino

🔍 Что значат две строки

  • servo.attach(9) — «серво воткнуто в пин 9», подготовить его.
  • servo.write(120) — «повернись на 120°». Можно от 0 до 180.
💡 setup и loop В Arduino setup() выполняется один раз при включении, а loop() повторяется без конца — прямо как игровой цикл в pygame!
🎮 Попробуй Поставь 0, потом 180 — увидишь крайние углы. Это пригодится, чтобы подобрать пределы суставов.
