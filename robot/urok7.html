Первый скетч

Цель урока Залить первый скетч на Nano и повернуть один сустав через драйвер PCA9685.
📚 Сначала поставь библиотеку В Arduino IDE: Инструменты → Управление библиотеками, найди «Adafruit PWM Servo Driver» и нажми «Установить». Она умеет общаться с PCA9685.

🧰 Arduino IDE — три шага

1. плата: Arduino Nano→ 2. выбрать порт→ 3. «Загрузить» ⬆

На Ubuntu порт у Nano обычно /dev/ttyUSB0 (у Nano стоит чип CH340). Если порт «серый» — см. урок 8 (права доступа и драйвер CH340).

📝 Скетч: один сустав на угол

#include <Wire.h>
#include <Adafruit_PWMServoDriver.h>
Adafruit_PWMServoDriver pwm;          // драйвер серво

void setup() {
  pwm.begin();
  pwm.setPWMFreq(50);            // серво работают на 50 Гц
  pwm.setPWM(0, 0, 500);        // канал 0 ~120° (повернуть)
}

void loop() { }
⬇️ one_servo_test.ino

🔍 Что значат строки

  • pwm.begin() и setPWMFreq(50) — подготовить драйвер (50 Гц).
  • pwm.setPWM(0, 0, 500) — канал 0, импульс длиной 500 (≈120°). Помни: ~150 = 0°, ~375 = 90°, ~600 = 180°.
💡 setup и loop В Arduino setup() выполняется один раз при включении, а loop() повторяется без конца — прямо как игровой цикл в pygame!
🎮 Попробуй Поставь 150, потом 600 — крайние углы. В основной прошивке мы будем писать понятные «градусы», а число импульса посчитается само.
